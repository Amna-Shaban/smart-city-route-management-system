# 🏙️ Smart City Route Management System

A high-performance C++ application designed to optimize urban transit routes. This system models a city graph network where intersections represent nodes and roads represent weighted edges, utilizing **Dijkstra's Algorithm** to compute the shortest path between locations.

## 🚀 Features
* **Graph Modeling:** Represents complex urban road networks with dynamic edge weights (representing distance or traffic).
* **Path Optimization:** Implements Dijkstra's algorithm to determine the absolute quickest route between two coordinates.
* **Console Visualization:** Clear, step-by-step route and distance tracking printed directly to the terminal.

## 🛠️ Technical Specifications
* **Language:** C++
* **Concepts Used:** Graph Theory, Shortest Path Routing, Greedy Algorithms, Data Structures (Min-Heaps / Priority Queues)
* **Development Environment:** VS Code

## 📁 Repository Structure
* `/src` - Contains the core execution logic and algorithm implementation.
* `/assets` - Visual mapping references and network route flowcharts.
