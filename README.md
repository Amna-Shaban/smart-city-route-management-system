# Smart City Route Management System

A Python-based web application that implements core Data Structures and Algorithms (DSA)—specifically **Dijkstra's Algorithm**—to find and optimize the most efficient paths between city routes. 

### 🛠️ Tech Stack
* **Backend:** Python
* **Frontend:** HTML5, CSS3 (Custom styling)
* **Core Concept:** Graph theory, routing algorithms, and path optimization.
